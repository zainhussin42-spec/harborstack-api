# HarborStack API

## Description
Simple Express API for managing crews and shifts.

## Setup
npm install

## Run
npm run dev

## Endpoints

### Crews
GET /api/v1/crews  
GET /api/v1/crews/:id  
POST /api/v1/crews  
PUT /api/v1/crews/:id  
DELETE /api/v1/crews/:id  

### Shifts
GET /api/v1/shifts  
GET /api/v1/shifts/:id  
POST /api/v1/shifts  
PUT /api/v1/shifts/:id  
DELETE /api/v1/shifts/:id
